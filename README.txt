Cidades do Mundo — Template estático pronto para deploy

Como usar:
1. Extraia os arquivos.
2. Faça upload para GitHub Pages, Netlify ou qualquer host de arquivos estáticos.
3. O site é responsivo e usa imagens dinâmicas do Unsplash (ex.: https://source.unsplash.com/1600x900/?tokyo).

Arquivos incluídos:
- index.html
- styles.css
- script.js

Personalização:
- Edite index.html para adicionar mais cidades (copie os blocos <article class="card">).
- Substitua as URLs das imagens por suas próprias imagens em /assets.
